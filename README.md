# VBA脱出 → 自分の地図を作る 12か月チャレンジ

Start: 2026-02-23
OS: Windows
Python: 3.14.3

## ルール
- ミニは必ず毎月クリア（これだけでOK）
- 月末に「一番動いた瞬間」を1行書く
- 完璧を目指さない（前進優先）

## 設計思想
前半6か月 = いろんな体験を少しずつする（何に興奮するか発見する）
後半6か月 = 一番好きだったものを深掘りする（6か月後に方向を決める）
